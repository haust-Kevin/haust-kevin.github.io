网站推荐

Github

优秀的开源网站

Stack Overflow

一个免费的编程问答网站，提供程序员之间的即时交流支持。

牛客竞赛

牛客竞赛OJ是专业的编程比赛和算法训练平台，包括ACM校赛、ICPC、CCPC、信息学奥赛、NOIP、NOI等编程比赛提高训练营。 面试题库 在线编程 面试 面试经验 AI 模拟面试 求职 学习

阿里巴巴矢量图标库

国内功能很强大且图标内容很丰富的矢量图标库，提供矢量图标下载、在线存储、格式转换等功能。


桌面应用
Visual Studio Code

vscode是一款免费开源的现代化轻量级代码编辑器甚至IDE

Snipaste

Snipaste 是一款免费无广告的截图、贴图软件，支持自动检测边缘、多种标注工具、二次编辑、图片质量提升等功能。

Everything

Windows文件夹搜索引擎，它能够基于文件名快速定文件和文件夹位置，将搜索效率提升100倍。

Clash for Windows

Clash是一个跨平台的基于规则的代理工具, 在网络和应用层运行, 支持各种代理和反审查协议的开箱即用。


移动应用
酷安

国内最大的数码社区网站

MT管理器

Android 平台文件管理 & 逆向修改神器

Magisk

Magisk是Android获取root的主流方式

LSPosed

LSPosed模块仓库是一个基于Xposed框架的模块管理平台

Clash for Android

Clash是一个跨平台的基于规则的代理工具, 在网络和应用层运行, 支持各种代理和反审查协议的开箱即用。

爱玩机工具箱

爱玩机工具箱是一款可以管理应用刷机，冻结，暂停，伪装，无障碍等功能的工具箱。

搜图神器

一款小而美但拥有海量图片资源的图片搜索神器，并支持以图搜图，拍照搜图。
